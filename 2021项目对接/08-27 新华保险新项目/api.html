获取token

基本信息

Path： /api/v1/xin-hua/token/getToken

Method： POST

接口描述：

注意：
1）请求
签名：算法为md5（联调时发算法代码）；数据为“clientId=客户端编号&secret=密钥&timestamp=时间戳”
例，md5(clientId=xxx&secret=xxx&timestamp=2021090811）
2）响应
成功返回json格式：
{ "token": "e5ac9c39669046d5b1fe8c70375a32b5", "expireTime": 7200}
错误返回json格式:
{  "success":1,  "msg": ""}
3）环境
